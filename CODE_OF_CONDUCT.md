# Code of Conduct

## Our Pledge

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus feedback on the work, not the person.
- Accept differing viewpoints and experiences.
- Give maintainers time to review issues and pull requests.

## Unacceptable Behavior

- Harassment, intimidation, or abusive language
- Personal attacks or derogatory comments
- Publishing private information without permission
- Spam, trolling, or repeated off-topic comments

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or restrict participation for behavior that violates this code of conduct.

To report a concern, contact the maintainer privately through the contact options on the maintainer's GitHub profile.
